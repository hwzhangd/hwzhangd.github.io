vim 个人总结

vim几个模式

  • normal Mode 默认进行vim的模式
  • visual Mode 可视模式
    • 普通模式下按v进行
    • ctrl + v 可视模式的块选择
  • insert Mode 插入模式
    • i 进入
  • select Mode 选择模式
    • gh进入
  • command-Line/Ex Mode 命令行模式
    • 单命令行 :
    • 多命令行 Q

map 映射

  • 命令组合
    • nore非递归
    • n 普通模式下生效
    • v 可视模式下生效
    • i 插入模式下生效
    • c 命令行模式下生效
  • Recursive Mapping 映射可递归
  • unmap 删除映射
  • mapclear 消除相关模式下的所有映射

辅助设置

vim设置

常用快捷键

  • 打开其他文件 :tabe path/source
  • 标签页中切换 gt
  • 分屏:sp :vsp
  • 分屏中切换ctrl + w + h/j/k/l 分别对左下上右
  • 进行关连文件 gf
  • 退回到原来的文件 ctrl + o
  • 折叠
  • 块上下跳转 %
  • 添加统一信息，例如注释
    ctrl + v -> ctrl + i -> // -> esc ,esc

  • 删除注释等
    ctrl + v -> d

  • 光标至行首shift + ^

  • 光标至行尾shift + $
  • 光标至行开头0
  • 光标移动 左下上右hjkl
  • page down ctrl + f
  • page up ctrl + b
  • 移动到文件尾G
  • 删除整行dd,删除N行dN
  • dw删除一个词，d$ 删除到行尾，d0删除到行首
  • dgg向上删除至第一行
  • dG向下删除至最后一行
  • u撤消
  • y,yn复制，p粘贴
  • w,b上下一个单词

命令

  • set:nu 开启行号
  • set:nonu 取消行号
  • set:diff 比较两个文件不同
